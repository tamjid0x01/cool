# Get All Urls  (COOl)
enum
Fetch known URLs from  , the Wayback Machine, and Common Crawl. Originally built as a microservice by [Tamjid ](https://www.linkedin.com/in/tamjid0x01/)

### usage:
```
▻ printf 'example.com' | cool
```

or

```
▻ cool example.com
```

### install:

```
▻ git clone https://github.com/tamjid70/cool && cd cool
▻ go build -o $GOPATH/bin/cool cool.go
```

or

```
▻ go get -u github.com/tamjid70/cool
```
